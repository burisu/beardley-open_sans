# Beardley::OpenSans

Supplies Open Sans fonts for JasperReports

## Installation

Add this line to your application's Gemfile:

    gem 'beardley-open_sans'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install beardley-open_sans
